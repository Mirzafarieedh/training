### Numerical Derivative Tool
- Built a function that takes a mathematical expression and a value of x
- Computes slope using numerical approximation: (f(x+h) - f(x)) / h
- Supports basic functions like sin, cos, log, etc.
- Demonstrates how derivatives can be computed programmatically

Example:
- Input: f(x) = x**2 at x = 2
- Output: slope ≈ 4

## Key Takeaways
- Hashing helps reduce time complexity
- Simple logic can solve common interview problems
- Gradient descent is based on slope (derivative)
- Brute force becomes inefficient as input grows
- Numerical derivatives enable computing slopes programmatically, forming the basis of optimization methods like gradient descent
